# 📝 Interactive To-Do List Website

A sleek, responsive, and user-friendly To-Do List website built to help users organize their daily tasks efficiently. This project showcases the practical application of frontend web development fundamentals.

## 🚀 Live Demo
You can view the live, working website here: 
👉 [https://dikshatehdpuriya.github.io/TO-DO-LIST/](https://dikshatehdpuriya.github.io/TO-DO-LIST/)

---

## ✨ Features
* **Add Tasks:** Quickly type and add new items to your list.
* **Complete Tasks:** Mark tasks as completed with a satisfying visual strike-through.
* **Delete Tasks:** Remove individual tasks when they are no longer needed.
* **Responsive Design:** Beautifully optimized to work on mobile phones, tablets, and desktop screens.

---

## 🛠️ Technologies Used
* **HTML5:** For creating the semantic structure of the web application.
* **CSS3:** For modern styling, layout positioning, and responsive design.
* **JavaScript (ES6):** For dynamic DOM manipulation, task logic, and interactive features.

---

```text
├── index.html   # Main HTML file for the page structure
```
